# src/

TypeScript/JavaScript consumers of the immutable tables.

Files:
- `index.ts`, `index.js`: entry points
- `core/`: core runtime
- `parser/`: binary/ternary parsing
- `types/`: TypeScript types
- `utils/`: utilities

This folder produces the compiled outputs used by:
- Cloudflare Worker runtime
- Node.js CLI consumers
- browser/worker surface
